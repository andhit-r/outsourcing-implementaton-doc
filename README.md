# Implementasi Penggajian Outsource — OpenSynergy Indonesia

Dokumentasi implementasi pengelolaan penggajian karyawan outsource dan invoicing klien menggunakan modul Odoo 14 SSI (PT. Simetri Sinergi Indonesia).

## Tentang Dokumentasi Ini

Dokumentasi ini ditujukan untuk **implementor** yang bertugas mengkonfigurasi dan mendeploykan sistem Odoo untuk perusahaan penyedia jasa outsourcing (vendor tenaga kerja).

## Membaca Dokumentasi

Dokumentasi ini di-deploy ke GitHub Pages. Akses di:
`https://<username>.github.io/outsourcing-implementation-doc`

## Menjalankan Dokumentasi Secara Lokal

```bash
pip install mkdocs-material
mkdocs serve
```

Buka di browser: `http://127.0.0.1:8000`
